# Game

A game is constructed from players, turns and rounds.

The game is concluded when each [player](player.md) has completed 13 rounds.

## New Game

A new game can be initiated that

- Aborts existing game
- Request Player Registrations, followed by
    - Initialiation of each [players score card](player-score-card.md)
    - Initialiation of first [round](rounds.md#round)
    - Initialiation of first player's [turn](rounds.md#turn)

## Players

### Player Registration

Each [player] sets their 'name', default for each possible player is
- Player One
- Player Two
- Player Three
- Player Four

Each player must indicate whether they are participating against one of the four [player](player.md) spots

### Players per game

Minumum number of [players](player.md) is 1
Maximumnumber of [players](player.md) is 4

## Rounds

The [round](rounds.md#round) is concluded when each [player](player.md) has concluded their [turn](rounds.md#turn)

### Rounds per game

The number of rounds is equal to the sum of [upper scorecard categories](player-score-card.md#upper-section) plus [lower scorecard categories](player-score-card.md#lower-section)

When the last round is concluded the game is over

## Turns

The turn is concluded when a player selects a [score card](player-score-card.md) category.

## Leader Board - Current Game

- Current game
- A list of player names and their [player score card](player-score-card.md) total
- Ordered by highest total score to lowest
- It is updated after each turn

## Leader Board - All Time

- Since records began
- A list of player names and their [player score card](player-score-card.md) total
- Ordered by highest total score to lowest
- Top 5 scores only
- It is updated after game is completed

## Leader Board - This Week

- Recorded in the last 7 days
- A list of player names and their [player score card](player-score-card.md) total
- Ordered by highest total score to lowest
- Top 5 scores only
- It is updated after game is completed

## Aborted Game

- Game is concluded
